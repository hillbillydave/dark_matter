## Overview

This repository contains the code, data, and reproducible workflows for the **Hybrid Capacity–Expansion Framework (v3.3)** — a unified geometric model in which dark–matter phenomenology arises from the *observer‑dependent realization of spacetime geometry*, not from new particle species.

The framework integrates three theoretical components:

- **ODIM** — the Observer‑Dependent Information Metric  
- **Quiet Scalar Time (QSTF)** — rate‑first emergent time  
- **Foundry Geometry** — the projection–capacity identity  

These combine into a single operational law:



\[
g_{00}^{(O)} = -\frac{\Pi_O}{E},
\]



where  
**$\Pi_O$** = projection capacity (informational coupling)  
**$E$** = expansion (environmental decoherence)

Dark‑matter–like behavior emerges when $\Pi_O/E$ enters a low‑capacity regime.
